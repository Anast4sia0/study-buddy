# StudyBuddy

## Overview
StudyBuddy is a student productivity web application developed using React, JavaScript, CSS, and Local Storage.

The application helps students organize their studies by creating study plans, managing tasks and subtasks, tracking progress, and using focus timers to improve productivity.

## Features

### User Authentication
User registration
User login
User logout
Input validation
Password validation

### Study Plan Management
Create study plans
Open study plans
Delete study plans
Store plans in local storage

### Task Management
Add tasks to study plans
Set task priorities
Delete tasks
Mark tasks as completed

### Subtask Management
Create subtasks
Complete subtasks
Delete subtasks

### Progress Tracking
Progress percentage calculation
Completed task tracking
Dashboard statistics

### Focus Timer
Task-specific study timers
Start, pause, and reset functionality

### User Features
Profile section
Settings section
Theme preferences
Dark mode
