# StudyBuddy

StudyBuddy is a React web application that helps students create study plans, manage tasks, track progress, and use a focus timer.

## Features

- User-friendly study planning interface
- Add and manage study tasks
- Track completed tasks
- View progress percentage
- Use a focus timer
- Save data in the browser
- Responsive design

## Technologies Used

- React
- Vite
- JavaScript
- HTML
- CSS
- localStorage
