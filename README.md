# StudyBuddy
A student productivity web application built with React, JavaScript and CSS.
StudyBuddy helps students organize their study plans, manage tasks, track their progress and stay focused while studying using a built-in timer.


## features

### User Authentication
User registration
User login
User logout
Form validation

### Study Plans
Create study plans
View study plans
Delete study plans

### Task Management
Add tasks
Delete tasks
Mark tasks as completed
Set task priorities

### Subtasks
Add subtasks
Complete subtasks
Delete subtasks

### Progress Tracking
Track completed tasks
Track completed subtasks
View study progress

### Focus Timer
Start the timer
Pause the timer
Reset the timer

### User Settings
Profile page
Settings page
Dark mode
Save user preferences

### data Storage

Uses Local Storage
Data remains available after refreshing the page
